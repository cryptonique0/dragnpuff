# Changelog

All notable changes to House of the DragNs will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-25

### Added
- Initial release of House of the DragNs
- DragNPuff ERC721 smart contract with governance voting
- ERC721Minter contract with dual-tier pricing
- Firebase backend with Cloud Functions
- Farcaster integration with frames and cast actions
- Mint, Choose, Leaderboard, and Breathe Fire frames
- Support for 7 Houses of the DragN
- Generative artwork with 111 unique traits
- $NOM token integration

### Security
- Implemented AccessControl for contract permissions
- Added EIP712 signature verification
- Rate limiting on API endpoints

### Documentation
- Comprehensive README
- Contract documentation
- Firebase setup guide
